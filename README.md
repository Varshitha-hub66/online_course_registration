# Online Course Registration System

## 📌 Project Description
A web-based Online Course Registration System developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap.

This project allows students to:
- Register and login
- Enroll in courses
- Manage profiles
- View enrollment history

Administrators can:
- Manage students
- Manage departments
- Manage sessions & semesters
- Add/Edit courses
- View reports

---

## 🚀 Features

### Student Module
- Student Registration
- Student Login
- Course Enrollment
- Enrollment History
- Profile Management
- Password Change
- Printable Enrollment Details

### Admin Module
- Admin Login
- Manage Students
- Add/Edit Courses
- Manage Departments
- Manage Sessions & Semesters

## 📂 Project Structure

```bash
onlinecourse/
│
├── admin/              # Admin panel files
├── assets/             # CSS, JS, Images
├── includes/           # Database connection files
├── studentphoto/       # Uploaded student images
├── sqlfile/            # Database SQL file
│
├── index.php           # Main login page

├── my-profile.php      # Student profile
├── enroll-history.php  # Enrollment history
Installation Steps
1.Install XAMPP or WAMP
2.Copy project folder into:
3.htdocs (XAMPP)
4.www (WAMP)
5.Start Apache and MySQL
6.Open phpMyAdmin
7.Create a new database
8.Import SQL file from sqlfile
9.Configure database credentials in includes
10.Run the project in browser
