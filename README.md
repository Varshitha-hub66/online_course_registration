# online_course_registration
Online Course Registration System

A web-based Online Course Registration System developed using PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap. This project allows students to register, enroll in courses, manage profiles, and view enrollment history, while administrators can manage students, departments, sessions, semesters, and courses through an admin dashboard.

Installation Steps
Install XAMPP or WAMP server.
Copy the project folder into:
htdocs (XAMPP)
www (WAMP)
Start Apache and MySQL.
Open phpMyAdmin.
Create a new database.
Import the SQL file from the sqlfile folder.
Configure database credentials in the includes folder if needed.
Open browser and run:
http://localhost/onlinecourse
